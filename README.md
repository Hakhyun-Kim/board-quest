# 판 위의 원정대 (Board Quest)

### ▶ [브라우저에서 바로 플레이](https://hakhyun-kim.github.io/board-quest/)

길을 골라 나아가고, 큰 판 위에서 **턴제로 겨루는 전술 원정** 게임.
히어로즈 오브 마이트 앤 매직의 갈림길 이동과, 어스토니시아 스토리·천사의 제국 같은 격자 턴제 전투를 한 판에 담았습니다.
자매작 **[백층 던전](https://github.com/Hakhyun-Kim/dungeon100)** 에서 파생 — 절차 생성·외부 에셋 0개 원칙을 그대로 잇되, 코어는 실시간 액션 대신 **턴제 전술**입니다.

## 게임 방법

- 🗺️ **원정 지도** — 갈림길을 골라 나아갑니다. ⚔️전투 · 🏘️마을 · 💰보물 · 🏕️야영 · 👑적장(마지막 단계).
- ⚔️ **전투** — 16×12 판 위의 턴제. 라운드마다 **속도 순으로 전원이 한 번씩** 행동합니다.
  - 한 턴 = **이동 + 행동 1회**(공격 / 회복 / 아이템 / 대기). 행동 전이면 이동을 취소할 수 있습니다.
  - **반격**: 때린 뒤 상대가 살아 있고 *상대의 사거리 안*이면 되받아 맞습니다 — 궁수로 멀리서 쏘면 안전하고, 근접에 붙으면 위험합니다.
  - **지형**: 숲은 방어 +2, 언덕은 공격 +2, 물·바위는 통과 불가. 자리 싸움이 곧 승부입니다.
  - 표적을 고르면 **예상 피해 · 처치 여부 · 반격량**을 미리 보여 줍니다.
- ✨ **특기** — 직업마다 고유 특기가 있고 **쿨다운**으로 돌아옵니다. 검사 🌀회전베기(인접한 적 전부), 궁수 🎯저격(사거리 +2·반격 없음), 사제 🙏기도(주변 아군 전체 회복), 창병 🔱관통(뒤에 선 적까지). 언제 쓰고 언제 아낄지가 전투의 갈림길입니다.
- 🧑‍🤝‍🧑 **원정대** — 검사·궁수·사제로 시작해 마을에서 동료를 영입합니다(최대 4명). 체력과 레벨은 전투 사이에 **이월**되므로, 언제 정비할지가 원정의 핵심 판단입니다.
- 🧪 **아이템** — 회복약 · 폭탄(범위) · 숫돌(공격 강화)을 전투 중 행동으로 사용합니다.

조작: **파란 칸을 누르면 이동, 붉은 적을 누르면 공격** — 따로 메뉴를 거치지 않습니다.
적을 **우클릭(모바일은 길게 누르기)** 하면 행동하지 않고 정보만 보고, 마우스를 올리면 예상 피해가 미리 보입니다.
회복·아이템·대기는 아래 메뉴에서 고르며, 메뉴는 **방향키 + Enter**로도 다룰 수 있습니다.

## 로컬 실행

```bash
npm install
npm run dev   # http://localhost:5176
```
요구 사항: Node.js 20+. `npm run build` 프로덕션 빌드, `npm run typecheck` 타입 검사.

## 기술 스택

React 18 · TypeScript · Vite · three.js · @react-three/fiber
백엔드 없음(localStorage) · **외부 에셋 없음** — 절차 생성 지오메트리, 시드 고정 난수(mulberry32) 기반 지도·보드 생성, Web Audio 합성 효과음 (폰트: [Jua](https://fonts.google.com/specimen/Jua), SIL OFL 1.1).

전투 로직은 렌더와 분리된 **순수 함수**(`src/lib/battle.ts`)라 헤드리스로 자동 검증·밸런스 측정이 가능합니다.
